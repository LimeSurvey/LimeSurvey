Adicionando uma Nova Pergunta
Clique neste ícone  para adicionar una nova pergunta ao grupo.

Podem-se adicionar quantas perguntas sejam necessárias em cada grupo. Quando se clica no ícone de "Adicionar Nova Pergunta", aparecerá uma tela onde se deverá adicionar um Código: este campo é obrigatório e se utiliza para determinar a ordem em que devem aparecer as perguntas dentro de um grupo. O código da pergunta pode ter até 5 caracteres. As perguntas são ordenadas alfabeticamente. O campo Pergunta: pode ser tão comprido como seja necessário e na realidade contém a pergunta que será apresentada ao participante. No campo de Ajuda: pode-se adicionar a mensagem de ajuda que aparecerá debaixo da pergunta e das respostas associadas. No campo Tipo: uma aparecerá uma lista que contém os diferentes tipos de perguntas disponíveis. Uma explicação detalhada destes tipos está nas instruções. Pode-se fazer que uma pergunta seja Obrigatória: para assegurar-se que o participante não passe a pergunta sem tê-la respondido. Em alguns tipos de perguntas poderá indicar um 'Sim' ou um 'Não' no campo Outro: Nesses casos, a pergunta terá uma resposta 'Outra' adicional.
